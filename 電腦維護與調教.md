## 更新完後，針對電腦的安全性調整
1. 可以檢查SELINUX
```
# getenforce
Permissive

# setenforce

未來環境需要，所以改成permissive
# vim /etc/selinux/config
  SELINUX=permissive
```

2. 關閉root給予外部連線的權限，還有把DNS偵測關閉
```
# vim /etc/ssh/sshd_config
PermitRootLogin no
UseDNS no

# systemctl restart sshd
```

3. 關閉不需要的埠口
可以用 netstat -tlunp 檢查

4. 修改防火牆設定
習慣使用iptables，所以把firewalld停掉，改使用iptables。
其餘設定看環境需要。可以寫成腳本執行。
